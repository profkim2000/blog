    postgis 설치 2/2

    pgAdmin에서 위와 같이

    Servers > PostgreSQL 16 > Databases > Postgres > Extensions를 오른쪽 클릭한 다음,

    Create > Extension을 선택한다.

    위와 같이 postgis를 선택하고 [save] 버튼을 누른다.
