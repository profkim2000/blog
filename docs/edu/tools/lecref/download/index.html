    lecref

    다운로드

    • 버전 1.1(2024. 10. 15) - Window amd64
    • 버전 1.0(2024. 9. 12) - Windows amd64

    실행방법

    이 프로그램은 별도의 설치파일이 없습니다.
    압축 풀고 바로 실행하면 됩니다.

    실행 시 아래와 같은 메시지가 나오면,

    [예] 버튼을 눌러주세요.

    그럼 microsoft의 어떤 페이지로 이동하는데, 여기서 프로그램을 다운로드한 후 설치하면 됩니다.

    * lecref 1.0 프로그램은 windows app sdk(WinUI 3)를 이용해 만들어 졌습니다. 다운로드받아 설치한 프로그램은 windows app sdk를 실행시키기 위한 프로그램입니다.

    문제해결

    만약 프로그램을 실행했는데 화면에 아무런 변화가 없으면 visual c++ 재배포 가능 패키지 파일을 설치하시면 됩니다.

    https://learn.microsoft.com/ko-kr/cpp/windows/latest-supported-vc-redist?view=msvc-170

    위 링크에서 파일을 다운로드받아 설치하시면 됩니다.
    내 PC의 아키텍처를 정확히 모른다면,

    대부분의 경우 위와 같이 x64 파일을 다운로드받아 설치하시면 됩니다.

    * 이 프로그램은 visual studio 2022에서 winui 3를 이용해 작성되었습니다.
