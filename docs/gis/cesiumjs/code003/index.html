    개발환경 구축 페이지와 같이 프로젝트 명을 cesium_test로 했고, 그렇게 생성된 프로젝트가 d:\cesium_js 폴더에 생성된 경우,

    다음과 같이 2개의 파일을 생성한다.

<body onload="render()">
  <div id="cesiumContainer"></div>
  <div id="toolbar"></div>
  <script>
    async function render()
    {
      Cesium.Ion.defaultAccessToken = "토큰";

      const viewer = new Cesium.Viewer("cesiumContainer", {
        terrainProvider: await Cesium.CesiumTerrainProvider.fromIonAssetId(1)
      });

      viewer.scene.globe.depthTestAgainstTerrain = true;

      try 
      {
        // 애셋을 추가한다.
        var tileset = await Cesium.Cesium3DTileset.fromIonAssetId(96188);
        viewer.scene.primitives.add(tileset);

        // 시계탑
        tileset = await Cesium.Cesium3DTileset.fromIonAssetId(시계탑ID);
        viewer.scene.primitives.add(tileset);

        // 또 다른 asset
        tileset = await Cesium.Cesium3DTileset.fromIonAssetId(애셋ID);
        viewer.scene.primitives.add(tileset);
      } 
      catch (error) 
      {
        console.log(error);
      }
    }
  </script>
</body>
</html>

    * 만약 아래의 과정을 코드 1에서 이미 했다면 그냥 브라우저에서 F5만 눌러주면 된다.

    command 창을 열고 다음과 같이 실행한다. 

    d:

cd d:\cesium_js

node server.js
